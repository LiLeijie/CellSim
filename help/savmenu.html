File|Save|Save similarity
   Save cell type similairy score.

File|Save|Save common network
   Save the common network which has a mark on the fourth column between two certain cells.
    If the two cells has common network, the table will show "Common Network" which can be double clicked to display the network.
    If only one cell has regulation network, the table will show the cells name which can be double clicked to display the network.
    If none of the two cells has common network, the table will show "No Network".
